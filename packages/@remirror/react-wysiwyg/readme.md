## Installation

```bash
# yarn

# pnpm
pnpm add @remirror/react-wysiwyg@next @remirror/pm@next @remirror/react@next

# npm
npm install @remirror/react-wysiwyg@next @remirror/pm@next @remirror/react@next
```
